# Shell Permissions Project

This repository contains my exercises on **Linux file permissions and user management** as part of the AI Academy course at DLH.

---

## 📘 Objective

To understand and practice how Linux handles:

* Users and groups
* File ownership
* File and directory permissions
* Permission modification using shell commands

---

## 🧩 Tasks Covered

### 👤 User & Identity

* `0-iam_betty` – Switch current user to **betty**
* `1-who_am_i` – Print current user
* `2-groups` – Show user groups

---

### 📁 File Ownership

* `3-new_owner` – Change owner of file
* `13-change_group` – Change group owner
* `14-change_owner_and_group` – Change owner & group recursively
* `15-symbolic_link_permissions` – Modify symbolic link ownership
* `16-if_only` – Conditional ownership change

---

### 📄 File Permissions

* `5-execute` – Add execute permission (owner)
* `6-multiple_permissions` – Set multiple permissions
* `7-everybody` – Add execute for all users
* `8-James_Bond` – Set specific permission pattern
* `9-John_Doe` – Apply custom permission mode
* `10-mirror_permissions` – Copy permissions from another file

---

### 📂 Directory Permissions

* `11-directories_permissions` – Add execute to directories only
* `12-directory_permissions` – Create directory with specific permissions

---

### 🧪 Basic File Operations

* `4-empty` – Create an empty file

---
